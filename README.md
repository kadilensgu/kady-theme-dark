# Kady 主题 · VS code Theme

kady-theme-dark

## 简介

一款温柔而清晰的 VS Code 暗色主题，专为专注编码与长久阅读设计。

色彩经过反复调试：低饱和蓝灰基底缓解视觉疲劳，柔和的语法高亮在保持可读性的同时不抢夺注意力；括号匹配、光标行、活动标签页等关键反馈细腻克制，让界面成为你思考的延伸，而非干扰。

支持所有主流语言语法高亮，兼容常见插件（如 Bracket Pair Colorizer、Peacock、GitLens）。已适配 VS Code 最新版 UI（包括侧边栏、编辑器、终端、设置面板等全场景）。

## 安装方式

1. 打开 VS Code
2. 进入扩展市场（Ctrl+Shift+X / Cmd+Shift+X）
3. 搜索 “Wenwen Theme”
4. 点击安装并启用

你也可以在设置中手动应用：

```
File > Preferences > Color Theme > Wenwen Theme
```

愿每次打开编辑器，都像回到安静而熟悉的地方。
